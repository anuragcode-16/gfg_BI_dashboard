# InsightAI - Data Intelligence Platform

An AI-powered web application that allows non-technical users to generate interactive data dashboards using natural language prompts. Simply type a question in plain English and get instant charts, insights, and visualizations.

## Problem Statement

Business users often lack technical skills like SQL or BI tools, creating bottlenecks in data-driven decision making. InsightAI bridges this gap by converting natural language questions into SQL queries, selecting the optimal chart type, and rendering interactive dashboards in real-time.

## Features

- **Natural Language to SQL** - Ask questions in plain English, get accurate SQL queries generated by AI.
- **Smart Chart Selection** - AI automatically picks the best visualization (bar, line, pie, scatter, table).
- **Interactive Charts** - Hover for tooltips, zoom, pan, and explore data visually with Plotly.
- **Follow-up Questions** - Refine your analysis conversationally (e.g., "Now filter that to 2021-22").
- **Custom CSV Upload** - Upload your own CSV file and start querying immediately; suggested inquiries dynamically update based on the uploaded data.
- **Hallucination Handling** - Gracefully refuses to answer questions outside the available data schemas.
- **Self-Correcting SQL** - Automatically retries and fixes failed queries based on database errors.

## Tech Stack

| Component | Technology |
|---|---|
| Frontend | Streamlit |
| Charting | Plotly Express |
| LLM | Llama 3.3 70B via OpenRouter |
| Database | SQLite (auto-generated from CSV) |
| Backend | Python |

## Installation

1. **Clone the repository**
```bash
git clone https://github.com/anuragcode-16/gfg_BI_dashboard.git
cd gfg_BI_dashboard
```

2. **Install dependencies**
```bash
pip install -r requirements.txt
```

3. **Set up API key**
Create a `.env` file in the root directory:
```
OPENROUTER_API_KEY=your_openrouter_api_key_here
```

4. **Run the application**
```bash
streamlit run app.py
```

## Default Dataset

The application comes pre-loaded with the **India Life Insurance Claims** dataset containing:
- 25 columns tracking claim lifecycles across Indian insurers
- Metrics including claims pending, intimated, paid, repudiated, rejected, and unclaimed
- Performance ratios for settlement efficiency and denial rates
- Financial years 2020-21 and 2021-22

## Evaluation

Run the accuracy evaluation framework manually:
```bash
python accuracy_eval.py
```

This tests:
- **Data Retrieval** - Correct SQL generation
- **Chart Selection** - Appropriate visualization type assignment
- **Hallucination Handling** - Proper rejection of unanswerable questions
